# Pandas
## Pandas 개념 및 활용
### Pandas
- Python 기반 데이터 분석 라이브러리
- 관계형, 레이블 데이터 처리에 최적화
- 빠르고 유연한 데이터 구조 제공
- 데이터 분석을 위한 고수준 API 제공
- Numpy 기반 -> 머신러닝 라이브러리와 높은 호환성
### Pandas에서 다룰 수 있는 데이터 타입
- Tabular Data (표 형태 데이터)
  - Ordered / Unordered Data (순서 있는 / 없는 데이터)
  - Time Series Data (시계열 데이터)
  - Matrix-like Data (행렬 형태 데이터)
    - 행, 열로 구성된 데이터(열마다 다른 타입 가능)
  - Observational / Statistical Data (관측 / 통계 데이터)
    - 레이블이 없더라도 유연하게 처리 가능
- 빠르고 유연한 데이터 구조 제공 -> 표 형태로 다룰 수만 있으면 처리 가능!
### Pandas에서 제공하는 Data Structure
- Series (1차원)
  - 한 줄짜리 데이터 목록, 모든 값이 동일한 데이터 타입
- DataFrame (2차원)
  - 각 열(column)의 데이터 타입이 다를 수 있음, 크기 자유롭게 조절 가능한 형태
### Pandas의 장점
- 결측치 처리 용이 : NaN 값으로 결측치 표현, dropna(), fillna() 등 다양한 결측치 처리 함수 제공
- 행/열 추가, 삭제 간편 : append(), drop() 등으로 쉽게 조작
- GroupBy 제공 : SQL처럼 그룹 단위 집계 가능
- 다양한 객체 변환 : 리스트, 딕셔너리, Numpy 등에서 DataFrame 생성 가능
- Merge/Join 지원 : merge(), join(), concat() 등으로 데이터 결합
- 파일 입출력 지원 : CSV, Excel, DB, JSON 등 다양한 포맷 읽기 /쓰기 가능
### 주요 구성 요소
- DataFrame, Series, Index
- Index : DataFrame, Series의 고유한 Key 값 객체
- Column : DataFrame의 열 이름을 나타내는 객체.

## 데이터 프레임 생성 및 조작
- read_csv() : CSV 파일에서 DataFrame 불러옴
- head(), tail() : DataFrame의 처음/마지막 n행 출력
- shape : DataFrame의 행과 열의 수 확인
- info() : DataFrame의 요약 정보 출력 (컬럼, 데이터 타입, 결측치 정보)
- describe() : 수치형 데이터의 요약 통계 제공
- value_counts() : 특정 컬럼의 값 개수 세어 출력 -> 행 기준 중복된 행 얼마나 있는지 확인
  - Null값 무시하고 결과 리턴 (기본 dropna=True)
- sort_values() : 특정 컬럼을 기준으로 DataFrame 정렬
### DataFrame의 컬럼명과 인덱스
- dataframe.columns : DataFrame의 컬럼명 확인
- dataframe.index : DataFrame의 인덱스 확인 -> RangeIndex(start=0, stop=10, step=1) 형태로 출력
- dataframe.index.values[0:idx] : 인덱스 값 배열 형태로 출력
### 새로운 컬럼 생성 및 수정
- dataframe['new_column'] = value : 새로운 컬럼 생성
- dataframe['existing_column'] = new_value : 기존 컬럼 값 수정
- -> 일괄적으로 할당됨
### 데이터 삭제
- dataframe.drop(labels=None, axis=0, index=None, columns=None, level=None, inplace=False, errors='raise')
- row 삭제 시 axis = 0, column 삭제 시 axis = 1
- row 삭제 시 axis 생략 가능
- inplace : False -> 원본 유지, 드롭된 DataFrame를 새롭게 반환 / True -> 원본 DataFrame에서 드롭된 결과로 변경(권장하지 않음)
  - 메모리 절약도 없고 내부적으로도 copy후 다시 대입이라 이점이 x. 반환값 다시 변수에 할당하는 방식 권장.
- labels : 삭제할 row/column의 이름 또는 인덱스 지정
- 여러 행 삭제할 경우 리스트로 삭제할 행 입력

## 데이터 인덱싱과 필터링
### [] 
- 단일 선택 : df['col1'] -> series 반환
- 다중 선택 : df[['col1', 'col2']] -> DataFrame 반환
- boolean indexing과 함께 사용 가능 : df[df['col1'] > 10] -> col1이 10보다 큰 행 선택
### loc[] / iloc[]
- loc[] : 라벨 기반 인덱싱, 행/열 이름(label/index)으로 접근, 함께 지정 시 df.loc[row_label, column_label] 형태로 사용
- iloc[] : 정수 기반 인덱싱, 행/열의 위치로 접근
### Boolean Indexing 
- 조건식에 따른 필터링
- -> DataFrame의 부분집합을 추출하는 데 사용

## 데이터 정제 및 전처리
### 데이터의 처리
- 데이터 수집이 완료되면 데이터를 모델에 넣기 전에 알맞게 가공하는 과정
- 분석 결과의 질을 높이기 위해 데이터 변형하는 과정
- 어떤 전처리를 적용하느냐에 따라 원본 데이터가 다르게 변형 -> 결과도 달라질 수 있음
### 데이터 처리 방법
- 데이터 형식 맞추기
- 빈 칸 채우기
- 데이터 열 추가 (연관 데이터 추가)
- 데이터 열 추가 (이상치 제거)
### missing data 처리
- isna() : 데이터프레임 또는 배열에서 결측값(NaN) 있는지 여부를 True/False로 반환
- fillna() : 결측값을 지정한 값으로 대체(채움)
- dropna() : 결측값이 있는 행 또는 열을 제거

## 데이터 처리와 집계
### 객체 생성
- Default Index인 RangeIndex을 사용한 객체 생성 
- datetime index를 사용한 객체 생성 -> dates = pd.date_range('2024-01-01', periods=5) : 2024-01-01부터 5일간의 날짜 생성
- 딕셔너리를 사용한 객체 생성 -> data = {'A': [1, 2, 3], 'B': [4, 5, 6]} : A열과 B열이 각각 1,2,3과 4,5,6인 데이터프레임 생성
### 데이터 확인
- to_numpy() : DataFrame을 Numpy 배열로 변환
- Transpose() : DataFrame의 행과 열을 바꾸는 메서드
### label을 활용한 데이터 선택
- slice를 이용하여 행 추출
  - df['A'] : A열 전체 선택
  - df[0:3] : 0~2번 행 선택
- 특정 행과 열에 있는 데이터 추출
  - df.at[dates[0], 'A'] : 0번행의 A열 데이터 추출. 가장 빠른 방법
  - df.loc[dates[0], 'A'] : 0번행의 A열 데이터 추출. at보다 느리지만 여러 행과 열을 동시에 선택할 때 유용
### reindex로 새로운 행/열 구성
- DataFrame을 원하는 인덱스/컬럼 순서로 재배치, 존재하지 않는 라벨을 추가하여 새루운 구조의 dataframe 생성 가능
- 새로운 인덱스/컬럼 구성, 필요한 구조만 선택
### sort_values()
- 오름차순 기본 (ascending=True)
- 내림차순 (ascending=False) 설정 필요
### unique()
- 컬럼 내 몇 건의 고유값 있는지 파악
- nunique() : 고유값의 개수 반환
### sum(), max(), min(), count()
- Dataframem, Series에서 집계(Aggregation) 수행
### groupby() 
- 데이터를 특정 컬럼 기준으로 묶은 후 해당 그룹에 대해 집계 연산 수행
- groupby() 수행 후 DataFrameGroupBy 객체 반환 -> 집계 연산 수행하여 결과 얻음
- 동일 컬럼에 대해 서로 다른 집계함수 적용 시 agg() 메서드 사용
  - df.groupby('Pclass')['Age'].agg(['max', 'min']) -> Pclass별 Age의 최대값과 최소값 계산
- 여러  컬럼에 대해 서로 다른 집계함수 적용 시 agg() 메서드 사용, 딕셔너리 형태로 전달
  - df.groupby('Pclass').agg({'Age': 'max', 'Fare': 'min'}) -> Pclass별 Age의 최대값과 Fare의 최소값 계산
### apply()
- lambda 식을 결합하여 데이터 일괄 가공
- data['is_Adult'] = data['age'].apply(lambda x: 'Adult' if x >= 18 else 'Child') -> age 컬럼의 값이 18 이상이면 'Adult', 그렇지 않으면 'Child'로 변환하여 is_Adult 컬럼에 저장
- 함수 정의해서 적용 가능
  ```python
    def categorize_age(age):
      if age < 18:
          return 'Child'
      elif age < 65:
          return 'Adult'
      else:
          return 'Senior'
    data['age_category'] = data['age'].apply(categorize_age) -> age 컬럼의 값에 따라 'Child', 'Adult', 'Senior'로 변환하여 age_category 컬럼에 저장
  ```

## 데이터 병합 및 변환
### 데이터 병합
- concat() : 여러 DataFrame을 하나로 연결하는 함수, axis=0(행 방향) 또는 axis=1(열 방향)으로 연결 가능
  - 행 기준 병합 시 NaN 값이 발생 가능
- merge() : SQL의 JOIN과 유사한 방식으로 DataFrame을 병합하는 함수, on 매개변수로 병합할 열 지정
  - how 매개변수로 병합 방식 지정 (inner, outer, left, right)
  - suffixes 매개변수로 겹치는 열 이름에 접미사 추가 가능
  - on 매개변수에 열 여러 개 지정해 여러 열 기준 병합 가능
- join() : DataFrame의 인덱스를 기준으로 병합하는 함수, on 매개변수로 병합할 열 지정 가능
  - how 매개변수로 병합 방식 지정 (inner, outer, left, right)
### 데이터 변환
- replace() : 원본 값을 특정 값으로 대체
  - data.replace({'old_value': 'new_value'}) -> DataFrame에서 'old_value'를 'new_value'로 대체
  - 여러 값 대체 시 딕셔너리 형태로 전달 가능
    - data.replace({'old_value1': 'new_value1', 'old_value2': 'new_value2'}) -> 'old_value1'을 'new_value1'로, 'old_value2'를 'new_value2'로 대체
  - 결측값 처리 시에도 사용 가능
- pivot() : 행과 열의 위치를 바꿔 원하는 표 형태로 재배치
  - 특정 기준을 열 방향으로 펼쳐서 만든 표 형태를 wide format이라고도 함
  - `df.pivot(index='row_label', columns='column_label', values='value_column')` -> row_label을 행으로, column_label을 열로 하여 value_column의 값을 채운 표 형태로 반환
- pivot_table() : pivot()과 유사하지만, 집계 기능이 추가된 함수, values 매개변수로 집계할 열 지정, aggfunc 매개변수로 집계 함수 지정
  - aggfunc 매개변수에 여러 집계 함수 지정 가능
    - `data.pivot_table(index='row_label', columns='column_label', values='value_column', aggfunc=['sum', 'mean'])` -> row_label을 행으로, column_label을 열로 하여 value_column의 합계와 평균을 계산하여 표 형태로 반환
- melt() : 여러 개의 열을 행 방향으로 펼쳐서 하나의 열로 모아주는 기능
  - 가로로 넓게 퍼져 있던 데이터를 세로로 길게 변환(long format) 하는 과정
  - 다양한 분석을 더 유연하게 하기 위해 변환이 필요한 경우 사용
