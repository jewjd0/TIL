## 오전 실습
### 모델 추론 평가 함수 evaluate()
- model.train(), model.eval() 모드가 따로 있어서 훈련 모드, 추론 모드를 설정해 줄 수 있다.
  - model.eval()을 하면 Dropout이 모든 뉴런을 다 켜고, BatchNorm이 미리 배워둔 전체 데이터의 평균/분산값 고정해서 사용. 
  - model.train()을 하면 Dropout이 무작위로 뉴런을 켜고 끄고, BatchNorm이 배치마다 평균/분산값 계산해서 사용. 
- 추론 모드에서는 with torch.no_grad()로 감싸주면 메모리를 절약할 수 있다. 미분 그래프를 치워 풀이 과정을 기록하지 않게 하는 것.
- 추론 모드에서는 optimizer.zero_grad(), backward(), step()를 사용하지 않음.
  - backward(), step()는 모델을 수정할 때 사용하는 함수 -> 추론에는 필요 없다

## CNN vs FCN
### 완전 연결층 (Fully Connected Layer, FCN)
- 입력 받아서 출력으로 변환하는 신경망의 기본 모듈
- 변환 예시
  - 이미지의 모든 화소 나열하여 1차원 벡터로 만들어 입력 데이터로 받는다
  - 출력 : 10 * 1의 1차원 벡터
  - 모딜 상수 : 모델이 학습해야 하는 파라미터
### 합성곱 레이어 (Convolutional Layer, CNN)
- 입력 이미지를 필터와 연산, 특징 맵(feature map) 뽑아내는 모듈
- 1차원 구조로 변환하는 FCN과 달리 3차원 구조 보존하면서 연산
- Convolution 연산 : 필터를 이미지 상에서 이동시키면서 내적 반복 수행, 내적으로 구한 모든 값을 출력 제공
  - 필터의 한가운데에 출력 값 삽입
- 차원 주의! filter는 항상 입력의 깊이(채널) 축과 동일한 차원
  - 3 * 32 * 32 입력 이미지에 3 * 5 * 5 필터 적용 -> 1 * 28 * 28 출력
- 필터의 이동 경로를 따라 모든 연산 값 모아 활성화 맵 출력
- 예 : 박스 필터
  - 3 * 3 * 3 필터 적용 -> 1 * 30 * 30 출력
  - 각 위치에서 필터의 화수 내 값의 평균 계산해 가운데에 저장, 전부 조회 후 출력
  - 패딩을 줘서 입력 이미지 가장자리에도 필터 적용 가능
- 필터를 여러개 써서 다양한 특징 맵 추출 가능
  - 3 * 32 * 32 입력 이미지에 10개의 3 * 5 * 5 필터 적용 -> 10 * 28 * 28 출력
- 출력 해상도 : 입력 해상도 - 필터 해상도 + 1
- 입출력 해상도 유지하고 싶을 시 출력값 중 정의되지 않은 경우, 0 혹은 가장 가까운 출력값으로 대체
- 필터는 일반적으로 홀수 크기 -> 중심 있어야 해서
