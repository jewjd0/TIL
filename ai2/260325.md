### .env 파일
- env 파일로 환경변수를 관리할 수 있다
- 민감 정보인 API 키 등을 코드에 직접 작성하지 않고, .env 파일에 저장하여 보안성을 높일 수 있다
- .env 파일은 프로젝트 루트 디렉토리에 위치해야 함.
### stride
- 모델 학습시 입력 데이터의 이동 간격을 나타내는 하이퍼파라미터
- 이미지 모델 학습시 필터가 원래는 1칸씩 이동하는데 스트라이드 지정하면 스트라이드만큼 이동.
- 추출되는 픽셀 수가 줄어들어 효율성 높아짐
- 출력 특징 맵 크기 : (입력 크기 - 필터 크기) / 스트라이드 + 1
- 예) 입력 크기 32, 필터 크기 3, 스트라이드 1 -> 출력 크기 (32 - 3) / 1 + 1 = 30
- 스트라이드가 2이면 출력 크기 (32 - 3) / 2 + 1 = 15.5 -> 15 (소수점 버림)
