## RAG 기반 Customer Service AI 개발 
### DOCKER
- 컨테이너 기반 가상화 플랫폼
- 포트, 볼륨, 환경변수 등 설정을 받아서 여러 사람이 같은 환경에서 개발할 수 있도록 해줌
- 새 환경에서 할 때 `docker compose up --build`
- 바로 시작하려면 `docker compose up`
- `docker compose down`으로 컨테이너 종료
- 재시작 시 `docker compose up -d`로 백그라운드 실행
- vscode 익스텐션에서 Container Tools 설치해서 컨테이너 내부에서 개발 사능

### retriever
- 질문과 가장 관련이 깊은 chunk를 찾아내는 모듈
- as_retriever()로 retriever 상태로 만들어서 변수에 저장해 사용 가능 
- search_kwargs={"k": 3}로 가장 관련이 깊은 chunk 3개를 찾아내도록 설정, 딕셔너리 안에 score_threshold: 0.5로 유사도가 0.5 이상인 chunk만 반환하도록 설정 가능
