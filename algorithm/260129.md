## 1차원 리스트
- 다양한 자료형의 요소들을 순서대로 저장하는 자료구조
- 여러 개의 변수 한번에 집어넣어서 관리 가능
- 별도 선언 방법이 없으면 변수에 처음 값 할당할 때 생성
```python
arr = list()
arr = []
arr = [1, 2, 3]
arr = [0] * 10  # 크기 10인 리스트 생성
```
- 인덱스로 요소 접근
```python
arr[0] = 10  # 첫 번째 요소에 10 할당
```
- 특징 
  - 순서 있는 자료형
  - iterable(순회가능), 직접접근가능
  - mutable(요소 변경 가능)
  - 정렬 보장 x
- 리스트와 배열은 다르다
- 순회
```python
for i in range(len(arr)):
    print(arr[i])
for i in arr:
    print(i)
```

### split()
- 문자열을 특정 구분자로 나누어 리스트로 반환
- 기본 구분자는 공백 문자
- immutable한 객체의 메서드는 주로 새 객체 반환
- 원본 객체를 손상할 수 없기 때문
### map()
- 리스트의 모든 요소에 특정 함수를 적용하여 새로운 리스트 반환
- 주로 입력값을 특정 자료형으로 변환할 때 사용
- 입력값 정수열을 정수 리스트로 반환할 때 주로 사용
- 엄밀히 말하면 map 객체 반환
- 리스트로 변환하려면 list()로 감싸야 함
```python
ai = list(map(int, input().split()))
```
- input().split()의 각 요소에 int 적용한 객체 반환
- 그걸 list()로 감싸서 리스트로 변환
### 강사님 참고
```python
for idx in range(len(ai)):
    ai[idx] = int(ai[idx])
```
- 배열 길이만큼 순회해서 각 요소를 정수로 변환
- 파이썬은 리스트에서 직접 변수 형변환해서 집어넣어도 된다. 타입 상관없이 리스트에 넣어도 되기 때문
- 이걸 간단화한게 map()
- 아웃풋 형식 중요하니까 꼭 맞추기
- 다담주 월요일 시험볼건데 출력 양식 틀리면 0점처리됨
- 디버깅 흔적 출력돼도 0점처리됨

## 2차원 리스트
- 리스트의 요소로 1차원 리스트를 가지는 자료구조
```python
list_2 = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
```
- 값 찾아내려면 순회해서 조회하기만 가능
- 2중 for문으로 조회
```python
for i in range(n):
    for j in range(m):
        print(list_2[i][j], end=' ')
    print()
```
- 세로 조회도 i, j 바꿔서 2중 for문 돌리면 됨
- 지그재그는 조건문으로 방향 바꿔가면서 조회
  - 연산줄이자고 별짓을다하는군
  ```python
  for i in range(n):
    for j in range(n):
      print(list_2[i][j+(m-1-2*j)*(i%2)], end=' ')
  ```
- 나중가면 numpy 쓸건데 일단 index개념잡기
