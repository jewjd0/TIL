## 1. JavaScript의 역사 & ECMAScript
- **ECMAScript**: Ecma International이 정의한 표준 스크립트 언어 명세
  - 브라우저마다 달랐던 JS 문법을 표준화
  - ES6(2015) 이후 문법이 주류
  - 타입스크립트: JS 기반, 타입 엄격
- 현재 주요 브라우저는 ECMAScript 표준을 따름

## 2. JavaScript 기본 문법

### 변수
- **식별자 규칙**: 문자, $, _로 시작 / 대소문자 구분 / 예약어 불가
- **표기법**


- **참조 자료형**: Object, Array, Function
  - 주소 참조, 가변, 주소 복사

### 연산자

- **동등 연산자(==)**: 암묵적 형변환, 사용 지양
- **일치 연산자(===)**: 값과 타입 모두 비교, 사용 권장
- **논리 연산자**: &&, ||, ! (단축 평가 지원)

### 조건문 & 반복문
- **while**, **for**: 기본 반복문
- **for...in**: 객체의 열거 가능한 속성 반복 (순서 보장 X)

## 3. DOM (Document Object Model)

### JS 실행 환경
1. HTML `<script>` 태그
2. .js 파일 (script src로 연결)
3. 브라우저 콘솔

### DOM 개념
- 웹 페이지를 구조화된 객체 트리(DOM Tree)로 제공
- 모든 요소/속성/텍스트는 객체, document 객체 하위로 구성
- DOM API: 요소 접근/조작 메서드 제공

### document 객체
- 웹 페이지 전체를 대표하는 객체
- DOM Tree의 진입점


### DOM 요소 선택
1. **document.querySelector()**: CSS 선택자, 첫 번째 일치 요소 반환
2. **document.querySelectorAll()**: CSS 선택자, NodeList 반환

## 4. DOM 조작

### 클래스 조작
- `element.classList.add('클래스')`: 클래스 추가
- `element.classList.remove('클래스')`: 클래스 제거
- `element.classList.toggle('클래스')`: 있으면 제거/없으면 추가

### 속성 조작


### HTML 콘텐츠/요소 조작
- `document.createElement('tag')`: 요소 생성
- `parent.appendChild(child)`: 자식 요소로 추가
- `parent.removeChild(child)`: 자식 요소 제거

## 5. 함수
- 참조 자료형, Function 객체
- **선언식**: function funcName() { ... }
- **표현식**: const funcName = function() { ... }
  - 표현식은 호이스팅 X, 선언 전에 사용 불가

### 매개변수
- **나머지 매개변수**: ...args (가변 인자)
- 매개변수/인자 개수 불일치 시: 누락 → undefined, 초과 → 무시

### Spread Syntax
- ... : 배열/문자열 등 반복 가능한 객체 전개

### 화살표 함수
- 짧은 함수 작성: `const arrow = name => 'hello,' + name`
- 콜백 함수 등에 자주 사용

## 6. 이벤트

### 개념
- 웹의 모든 동작(클릭, 입력, 스크롤 등)은 이벤트로 감지

### 이벤트 핸들러
- `.addEventListener('type', handler)`
  - 특정 이벤트 발생 시 콜백 함수 실행

### 버블링
- 이벤트 발생 시, 해당 요소 → 부모 → 조상 순으로 이벤트 전파
- **currentTarget**: 핸들러가 연결된 요소 (this)
- **target**: 실제 이벤트 발생 요소 (항상 동일)

### 기타
- **.preventDefault()**: 기본 동작(예: form 제출, 링크 이동) 방지
- **stopPropagation()**: 이벤트 전파 중단 (참고)
